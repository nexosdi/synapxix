🧠 Visión del producto
Synapxix es una plataforma educativa cognitiva que usa juegos adaptativos para evaluar y entrenar capacidades cognitivas de estudiantes. La IA analiza resultados de juegos en tiempo real y genera feedback pedagógico personalizado. Los docentes construyen roadmaps de aprendizaje; los alumnos avanzan a través de sesiones gamificadas con economía de créditos y XP.
✅ Estado general
El proyecto está en desarrollo activo con arquitectura monorepo Nx consolidada. La integración de IA vía Gemini está operativa con streaming SSE. El motor de juegos, el motor económico y el dashboard del alumno están implementados. CI/CD con GitHub Actions corre en cada push a main.

Riesgos técnicos
Bypass JWT en producción
El módulo evaluative tiene un bypass JWT para dev que puede llegar a producción por error.
→ Acción: flag de entorno + test que falle si NODE_ENV=production y bypass activo
Google AI API quota en streaming
El streaming SSE puede generar muchas llamadas a Gemini. Sin rate limiting por usuario, una sola sesión puede agotar la cuota.
→ Acción: throttle por userId en SseStreamService + monitoreo de cuota
package-lock.json divergente entre ramas
Conflictos frecuentes en package-lock.json al mergear. Evidenciado hoy mismo al hacer pull.
→ Acción: usar npm ci en CI, agregar package-lock.json a la estrategia de merge (.gitattributes)
Dependencia transitiva ng2-charts / Angular CDK
Requiere --legacy-peer-deps. Puede romper el build sin aviso al actualizar Angular.
→ Acción: evaluar migración a ngx-charts o esperar compatibilidad ng2-charts@11
Redis sin persistencia en dev
El caché de prompts IA (Redis) se pierde al reiniciar en local. No afecta producción si se configura con AOF.
→ Acción: documentar configuración Redis en docker-compose para dev persistente
